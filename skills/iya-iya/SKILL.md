# 使い方

「イヤイヤ」っぽく左右に振る短い動き。

実行:
python /home/pi/01/software/skills/iya-iya/scripts/iya_iya.py

振り幅（度）:
python /home/pi/01/software/skills/iya-iya/scripts/iya_iya.py --swing-deg 25

回数:
python /home/pi/01/software/skills/iya-iya/scripts/iya_iya.py --cycles 4

速さ調整（数値を大きくするとゆっくり）:
python /home/pi/01/software/skills/iya-iya/scripts/iya_iya.py --move-ms 300
